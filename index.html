TMall.com 字体使用方法

HTML结构：

            <i class="icons icon-ui"> &#x343a; <i>
            <i class="icons icon-ui"> &#x3448; <i>
            <i class="icons icon-ui"> &#x3449; <i>
            <i class="icons icon-ui"> &#xf012b; <i>
            <i class="icons icon-ui"> &#xf0148; <i>

CSS样式：

            /* here is icons basic style */
            .icons{
              display:inline-block;*display:inline;*zoom:1;
              font-style: normal;
            }
            
            /* you can use css3's @font-face property to set yourself web font */
            @font-face{
              font-family: mui-global-iconfont;
              src:url(fonts/font_icons_ui.eot);                                       /* IE9*/
              src:url(fonts/font_icons_ui.eot?#iefix) format('embedded-opentype'),    /* IE6-IE8 */
                  url(fonts/font_icons_ui.woff) format('woff'),           /* chrome、firefox */
                  url(fonts/font_icons_ui.ttf) format('truetype'),        /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
                  url(fonts/font_icons_ui.svg#iconfont) format('svg')     /* iOS 4.1- */
              }
            .icon-ui{
              font-family: mui-global-iconfont !important;
              font-size: 22px;
              color: #c40000;
            }
